# Kawu Install

Zip d’installation pour **Kodi** : `plugin.program.kawuinstall`.

## Installation

1. Installe [Kodi](https://kodi.tv/download)
2. Active **Sources inconnues** dans Kodi (Paramètres → Système → Modules complémentaires)
3. Télécharge le zip depuis la [dernière release](https://github.com/ztvplusfr/kawuapp/releases/latest)
4. Kodi → Extensions → Installer depuis un fichier zip → sélectionne `plugin.program.kawuinstall-….zip`
5. Ouvre **Kawu Install** (Add-ons → Programmes)
6. Entre le **code Discord** (`DL-XXXX`) → télécharge et installe le plugin Kawu
7. Ouvre **Kawu** (Vidéo) et entre ta **licence** (`XXX-XXX`, ex. `C8E-42B`)

Code + licence : Discord **@enrikkuu**.
